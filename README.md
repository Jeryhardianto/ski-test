# SKI-TEST

## Setup project
```
 1. git clone https://github.com/Jeryhardianto/ski-test.git ski-test
 2. cd ski-test
 3. composer install
 4. cp .env.example .env [setup database]
 5. php artisan key:generate
 6. php artisan migrate --seed
 8. open new terminal in directory ski-test or split terminal
 9. npm install
```
## How to run
```
 1. php artisan serve
 2. npm run dev
```
